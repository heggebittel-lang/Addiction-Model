## 📖 Overview

This repository contains the **initial preprint (v1.0)** of the working paper. It establishes the original conceptual framework and preliminary mathematical derivation for an extended **Rational Addiction Model**.

##  Key Innovation: The Exit Mechanism

The core contribution of this research is the introduction of an **Exit Mechanism ($e_t$)**, defined as an endogenous **hedging commodity**.

Unlike traditional models where addiction is often an irreversible path, this framework proposes:

1.  **Hedging Property:** Agents invest in the exit mechanism to counteract the marginal utility reinforcement of the addictive stock ($S_t$).
2.  **Soft Constraint:** The mechanism acts as an internal stabilizer to prevent the "utility explosion" often found in divergent addictive paths.
3.  **Intertemporal Stability:** It ensures the existence of a stable steady-state equilibrium through active risk management.
